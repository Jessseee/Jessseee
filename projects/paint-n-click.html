Paint 'n Click

A puzzle game with a randomly generated maze. Find your way through the maze by physically drawing the key objects to solve the puzzles.

To recognize doodles drawn by the player a TensorFlow model is trained, a visualization of the different layers can be seen bellow. The code to train the model can be found in train_model.py.

We used the Quick Draw bitmaps dataset to identify doodles with TensorFlow:
QuickDraw bitmap dataset.

We used the following tutorial for the TensorFlow training model:
TensorFlow recurrent QuickDraw tutorial.

For the input of the trained TensorFlow model we use OpenCV to resize a webcam input to 28x28 grayscale. as you can see in drawing_camera.py
